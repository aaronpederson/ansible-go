# ansible-go

Go is an open source programming language that makes it easy to build simple, reliable, and efficient software.

Tunables
--------
* None

Dependencies
------------
* [colstrom.apt-repository](https://github.com/colstrom/ansible-apt-repository/)

Example Playbook
----------------
    - hosts: servers
      roles:
         - role: colstrom.go
